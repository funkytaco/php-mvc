PDO examples for Postgres and MySQL are included. Mock data is used in this example & rendered via Mustache.

To use PDO instead of the Mock PDO

  • In Bootstrap.php uncomment $conn = $injector->make('\Main\PDO'); . It must stay below the $injector->define for PDO.
  • Your controller must use \Main\PDO; and not use \Main\Mock\PDO;
  • Config.php should have your database settings. $dbtype should be set to mysql or postgres. You can add other types supported by PDO, as this is just a PDO instantiation.
  • Stub out your database queries. In this demo, we stubbed them out to \Main\Traits\QueryData.php which is included by the PDO class, so $conn->getUsers() is treated like a local class function.
