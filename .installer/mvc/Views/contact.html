This is a template taken from http://getbootstrap.com/examples/theme/. Use it as a starting point to create something more unique by building on or modifying it.

As for this PHP-MVC framework, by default we use:

  • Bootstrap for front-end CSS.
  • Composer for dependency management and project setup
  • whoops for error handling
  • Klein for routing
  • mustache for templating
  • Auryn for dependency injection
